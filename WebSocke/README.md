
WebSocket 就是让浏览器和服务器之间可以随时互相说话的技术，不用每次都重新建立连接，特别适合需要实时交互的应用。

# Linux C语言实现WebSocket编程示例，该项目已经经过实际测试，能够在Linux操作系统上直接编译并运行。

## 使用说明

1. **环境要求**：确保您的开发环境为Linux操作系统，并安装有必要的编译工具，如gcc等。
2. **编译**：进入项目目录，使用`make`命令进行编译。
3. **运行**：编译成功后，执行生成的可执行文件，即可启动WebSocket服务器。
4. **测试**：您可以使用浏览器或者WebSocket客户端工具测试服务器的响应。

cd ~/bc6d9-main 
make
./websocket
websocat ws://localhost:9999
